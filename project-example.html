Atlas — trip planning, reimagined

A collaborative itinerary tool built for groups who can never agree on a plan.

RoleProduct design & frontend
StackReact, Node, Postgres
Timeline10 weeks, 2025
ClientIndependent release

The problem

Every group trip starts the same way: a spreadsheet, a group chat, and forty half-decided plans. Atlas began as an internal tool to fix our own trip-planning chaos before it became a public release.

The best itinerary tool isn't the one with the most features — it's the one nobody has to be talked into using.

What we built

Atlas combines a shared calendar, a lightweight voting system for activities, and a map view that updates in real time as the group commits to plans. Replace this paragraph with your own project narrative — what you tried, what changed, and why.

Design decisions

  • Every action is reversible, since group decisions change constantly.
  • Voting is anonymous by default, to avoid social pressure skewing choices.
  • Offline support, because trip planning happens on spotty airport wifi.
